# AIlover - 原神虚拟恋人

## 📖 项目简介
**AIlover** 是一款基于大语言模型（LLM）架构的“AI+网络应用”——**原神专属虚拟恋人平台**。
在这里，你可以跨越次元壁，与喜爱的原神角色进行深度、连贯且极具角色代入感的自然语言对话。本项目不仅是一个多元的聊天室，更是一个结合了爬虫技术、提示词工程（Prompt Engineering）与现代 Web 全栈技术的 AI 落地实践。

## 🚀 核心技术亮点


- **🕷️ 自动化领域知识抽取**：自研 Scrapy 分布式爬虫（`genshin_spider`），自动化采集原神 Wiki 上的角色故事、背景设定、技能数据与美术资源（立绘等），为 AI 构建详实可靠的“角色专属百科库”。
- **🧠 角色人格数字克隆**：摒弃千篇一律的机械回复，基于提取到的背景数据，利用高阶 **System Prompt 工程**将角色的性格、语言习惯和特殊羁绊精准注入 LLM。无需高昂的模型微调（Fine-tuning）成本，即实现了高还原度、强沉浸感的情感交互。
- **🧩 智能上下文记忆系统**：支持会话历史（Conversation History）的按需裁剪与长级记忆管理，让虚拟恋人能“记住”与你的点滴过往，聊得越多越懂你。
- **⚡ 现代化高并发应用架构**：
  - **后端**：采用 Python `FastAPI` 异步框架，支持高并发 AI 接口调用与流式响应（SSE），并内置了完善的鉴权与速率限制（Rate Limiting）。
  - **前端**：使用 `React` + `Vite` + `TypeScript` 打造，结合现代 CSS 方案还原沉浸式游戏 UI。

## 🛠️ 技术栈

- **AI 大模型侧**: LLM API (兼容 OpenAI / 国产大模型等规范)、Prompt Engineering
- **应用后端**: Python 3.11.4, FastAPI, Pydantic (数据校验)
- **数据采集侧**: Scrapy, BeautifulSoup4
- **客户端前端**: React 18, Vite, TypeScript, TailwindCSS / Vanilla CSS

## 📂 项目模块速览

- `/genshin_spider` - 自动化数据爬虫引擎，用于构建人物知识库与本地抓取。
- `/backend` - 高性能 FastAPI 服务器，处理对话逻辑、大模型交互与会话记忆。
- `/frontend` - 现代化 React 单页应用，实现沉浸式聊天体验。
- `/data` - 解析后的角色核心 JSON 数据及静态资源池。
- `/scripts` - 自动化的数据清洗、校验及部署脚本。
