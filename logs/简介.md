简介： 针对通用大语言模型在角色扮演场景下极易产生幻觉及人设崩塌（OOC）的痛点，设计并构建了一套基于 LLM 架构的垂直领域陪伴型 AI Web 应用（原神专属虚拟恋人平台）。实现了从领域知识自动化采集、专属 Prompt 封装调优，到前后端流式高频交互的全链路闭环。

技术栈：Python, FastAPI, React, TypeScript, Vite, Scrapy, LLM API, Pydantic, TailwindCSS

大语言模型幻觉与人设约束优化： 针对 AI 回复机械化与人设漂移问题，放弃高成本的模型微调（Fine-tuning），重构为Scrapy 自动化百科爬取 + 结构化高阶 System Prompt 注入的 RAG 思想处理模型，精准约束 LLM 发散边界，以零训练成本实现高还原度的零 OOC 角色情感交互。

长文本记忆与流式并发调优： 针对多轮长序列对话导致 Token 暴涨与高延迟瓶颈，后端定制滑动窗口 (Sliding Window) 裁剪 + 记忆归纳的会话长短时记忆管理策略；结合 FastAPI 异步并发与全面接入 SSE (Server-Sent Events) 流式通讯，有效解除 I/O 阻塞，将首字生成（TTFB）响应时间压降至毫秒级。

现代化前后端架构演进： 引入 React 18 + Vite 搭建丝滑单页应用（SPA），配合现代 CSS 方案还原沉浸式游戏 UI（打字机交互、角色生日日历等）；后端独立封装 Rate Limiting 防刷限流机制与会话鉴权，保障高并发接口安全性。

工程化与自动化数据治理： 编写系列 Python 自动化清洗脚本，实现对千万级字符的爬虫落地数据与本地图片资源的自动校验、修复及格式化；编写跨平台运行脚本实现前后端服务一键联调与本地交付。
